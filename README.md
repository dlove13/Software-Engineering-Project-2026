About the Project:
Arithmetic Expression Parser component for EECS 348 Software Engineering. The project is implemented in C++

Functional Criteria:
Takes an arithmetic expression input, parses it, and correctly solves using PEMDAS order of operations. It should return a double. Specific error messages are displayed in the event of a failed run
